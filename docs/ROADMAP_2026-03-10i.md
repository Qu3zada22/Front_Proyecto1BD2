## FIX-02 [ALTA] — Reescribir `topRestaurantes` sobre `resenaModel`

**Archivos:**
- `apps/api/src/reportes/reportes.service.ts`
- `apps/api/src/reportes/reportes.service.spec.ts`

**Nuevo pipeline (sobre `resenaModel`):**
```typescript
async topRestaurantes(limit = 10): Promise<any[]> {
    return this.resenaModel.aggregate([
        // 1. $match en resenas — usa índice restaurante_calificacion (IXSCAN)
        { $match: { activa: true, restaurante_id: { $exists: true } } },
        // 2. $group — promedio real desde fuente de verdad
        {
            $group: {
                _id: '$restaurante_id',
                avg_calificacion: { $avg: '$calificacion' },
                cantidad_resenas: { $sum: 1 },
            },
        },
        // 3. $match — mínimo 5 reseñas (no puede ir antes, necesita el conteo del $group)
        { $match: { cantidad_resenas: { $gte: 5 } } },
        // 4. $sort / $limit
        { $sort: { avg_calificacion: -1, cantidad_resenas: -1 } },
        { $limit: limit },
        // 5. $lookup — trae nombre, categorías, ubicación del restaurante
        {
            $lookup: {
                from: 'restaurantes',
                localField: '_id',
                foreignField: '_id',
                pipeline: [
                    { $match: { activo: true } },
                    { $project: { nombre: 1, categorias: 1, ubicacion: 1, _id: 0 } },
                ],
                as: 'restaurante',
            },
        },
        // 6. $unwind — convierte array de lookup a objeto
        { $unwind: '$restaurante' },
        // 7. $project — redondea promedio a 2 decimales
        {
            $project: {
                nombre: '$restaurante.nombre',
                categorias: '$restaurante.categorias',
                ubicacion: '$restaurante.ubicacion',
                avg_calificacion: { $round: ['$avg_calificacion', 2] },
                cantidad_resenas: 1,
                _id: 0,
            },
        },
    ]);
}
```

**Cambio en spec:** Reescribir el `describe('topRestaurantes')` para usar `mockResenaModel.aggregate` (ya existe en los providers del test) en lugar de `mockRestauranteModel.aggregate`. Verificar:
1. `$match` en stage 0: `{ activa: true, restaurante_id: { $exists: true } }`
2. `$group._id === '$restaurante_id'`
3. `$group.avg_calificacion = { $avg: '$calificacion' }`
4. Segundo `$match: { cantidad_resenas: { $gte: 5 } }`
5. `$lookup.from === 'restaurantes'`
6. `$unwind` presente
7. `$limit` con el argumento
