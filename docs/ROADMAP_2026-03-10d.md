## FIX-01 — Zero-padding de mes en `ingresosPorRestaurantePorMes`

**Prioridad:** LOW
**Archivo:** `apps/api/src/reportes/reportes.service.ts`

**Cambio en el `$project` stage:**
```typescript
// ANTES:
periodo: {
    $concat: [
        { $toString: '$_id.anio' },
        '-',
        { $toString: '$_id.mes' },
    ],
},

// DESPUÉS:
periodo: {
    $concat: [
        { $toString: '$_id.anio' },
        '-',
        {
            $cond: [
                { $lt: ['$_id.mes', 10] },
                { $concat: ['0', { $toString: '$_id.mes' }] },
                { $toString: '$_id.mes' },
            ],
        },
    ],
},
```

**Actualización de spec (`reportes.service.spec.ts`):**
- Actualizar el test de `ingresosPorRestaurantePorMes` para verificar que el
  `$project` incluye el `$cond` para zero-padding.
