## FIX-02 (OPCIONAL) — Limitar array de direcciones a 10 por usuario

**Prioridad:** LOW
**Archivo:** `apps/api/src/usuarios/usuarios.service.ts`

```typescript
// Opción A: usar $slice en $push
async addAddress(id: string, address: any): Promise<UsuarioDocument> {
    const updated = await this.usuarioModel
        .findByIdAndUpdate(
            id,
            { $push: { direcciones: { $each: [address], $slice: -10 } } },
            { new: true },
        )
        .select('-password')
        .exec();
    if (!updated) throw new NotFoundException('Usuario no encontrado');
    return updated;
}
```
