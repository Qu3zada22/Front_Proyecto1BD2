### FIX-02 — Implementar transacción ACID "Cancelar Restaurante"

**Prioridad:** HIGH
**Archivos:**

- `apps/api/src/restaurantes/restaurantes.service.ts` — agregar `cancelarRestaurante(id)`
- `apps/api/src/restaurantes/restaurantes.module.ts` — agregar MenuItem, Orden schemas + Connection
- `apps/api/src/restaurantes/restaurantes.controller.ts` — agregar `PATCH /:id/cancel`

La transacción:

```typescript
async cancelarRestaurante(id: string) {
  const session = await this.connection.startSession();
  session.startTransaction();
  try {
    const restaurante = await this.restauranteModel
      .findByIdAndUpdate(id, { $set: { activo: false } }, { new: true, session });
    if (!restaurante) throw new NotFoundException('Restaurante no encontrado');

    await this.menuItemModel.updateMany(
      { restaurante_id: restaurante._id },
      { $set: { disponible: false } },
      { session }
    );

    const histEntry = { estado: 'cancelado', timestamp: new Date(), nota: 'Restaurante cancelado' };
    await this.ordenModel.updateMany(
      { restaurante_id: restaurante._id, estado: { $in: ['pendiente', 'en_proceso', 'en_camino'] } },
      { $set: { estado: 'cancelado' }, $push: { historial_estados: histEntry } },
      { session }
    );

    await session.commitTransaction();
    return { cancelado: true, restaurante_id: id };
  } catch (err) {
    await session.abortTransaction();
    throw err;
  } finally {
    await session.endSession();
  }
}
```

Endpoint: `PATCH /api/restaurants/:id/cancel`

También actualizar spec: `restaurantes.service.spec.ts` con tests para la nueva transacción.
